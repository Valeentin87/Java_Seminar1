2. Код будем писать в папке java, создадим новый класс, назовём его Lesson1
3. В Java обязательно в коде указывать место входа в программу, так нызываемый метод main.
Для этого набираем сокращенно main (либо pvsm) и формируется точка входа в программу
public static void main(String[] args) {
    }
args - имя массива строк String[], к которому можно будет обращаться далее
4. Переменные
int i;  // объявление переменной
// int 32, long 64, short 16, byte 8
i = 0; // инициализация переменной
char j = 'h'; // хранит 16 битную информацию о номере символа, а также может хранить числа до 16 бит
char j = 45; // имеет место
char j = -45; // недопустимо
boolean r = true; // булевая переменная
String msg = "hello"; // является отдельным классом, включает в себя много методов
// String обозначаются в двойных кавычках "   ", буквы в одиночных '   '
r = 7 // нельзя так изменить тип переменной, заранее объявленный как булевый
float f = 4.65f;
double d = 3.45;
// по умолчанию целочисленные типы принимают значение 0, булевые false,
boolean[] bArray = new boolean[6] // объявление массива 6 булевых значений, изменять количество нельзя
